# Zurich Scout Mini Brand Guide

Zurich Scout is the working brand for the Good Boys customer-scouting automation agent. The brand should feel precise, calm, and enterprise-ready: a tool that helps Zurich qualify the right accounts before distribution or underwriting teams spend time on them.

This guide is for internal design consistency only. The current submission package now uses the Zurich Scout mark as a product brand cue.

## Brand Idea

**Positioning:** Account scouting, qualified.

**Core promise:** Zurich Scout turns broad market signals into a focused Zurich validation queue.

**Personality:**

- Precise, not flashy.
- Calm, not playful.
- Analytical, not cold.
- Enterprise credible, not generic AI.
- Broker-aware, not aggressive outbound.

## Selected Mark

Selected concept:

The mark works because it combines three useful metaphors:

- **Lens:** focused account review.
- **Radar:** active market sensing.
- **Gold signal:** one qualified opportunity emerging from noise.

Avoid describing it as a shield, target weapon, surveillance mark, or Zurich logo. It should read as a product icon for qualification, not protection or security.

## Color System

| Role | Hex | Use |
| --- | --- | --- |
| Zurich Blue | `#003399` | Primary product color, key headings, icon arcs |
| Deep Navy | `#02296A` | Titles, high-emphasis text |
| Signal Gold | `#D6A21E` | One small highlight: selected account, signal dot, active state |
| Cool Gray | `#5D6676` | Supporting text and metadata |
| Line Gray | `#D9E0EC` | Borders, dividers, quiet UI structure |
| Surface | `#F6F8FB` | Background bands and neutral panels |
| White | `#FFFFFF` | Main canvas and icon tile |

Use blue and white as the dominant impression. Gold should be rare: one accent per view is enough.

## Typography

Use a clean enterprise sans-serif. Preferred hierarchy:

- **Display / slide titles:** Helvetica Neue, Arial, Inter, or similar, bold.
- **Body:** Helvetica Neue, Arial, Inter, or similar, regular.
- **Tables / technical evidence:** same family, slightly tighter spacing, no decorative font.

Tone should be compact and factual. Avoid marketing-heavy adjectives.

## Icon Usage

Preferred use:

- App tile.
- Product thumbnail.
- Optional cover or appendix brand cue.
- Small internal prototype identifier.

Avoid:

- Placing it beside Zurich's logo as if it were an official Zurich product.
- Using it in the current final submission unless the deck is redesigned around it.
- Adding shadows, bevels, extra dots, shields, or complex networks.
- Recoloring the gold dot red unless it means an explicit alert state.
- Putting text inside the icon.

Minimum clear space: keep at least one gold-dot diameter around the mark.

## Visual System

Zurich Scout screens and slides should use:

- White canvas.
- Thin Zurich-blue top rule.
- Large, direct titles.
- Sparse cards with 1px borders.
- Numbers and action gates over decoration.
- Simple flow diagrams: `External universe -> Zurich Scout -> Zurich gates -> Action queue`.

Do not use:

- Decorative gradients as the main identity.
- Purple/blue startup gradients.
- Mascots.
- Shield shapes.
- Dark cyber-style dashboards.
- Dense node graphs.
- Salesy funnel imagery.

## Messaging

Primary message:

> Not more leads. Better pursuit decisions.

Supporting messages:

- Account scouting, qualified.
- Qualify first, activate after Zurich clearance.
- Public signals plus Zurich relationship memory.
- Pursue, watch, or reject with reasons.
- Broker-safe account review before outreach.

Words to use:

- qualify
- validate
- evidence
- account review
- Zurich gates
- broker ownership
- white space
- relationship memory

Words to avoid:

- scrape
- blast
- autonomous outreach
- lead bot
- surveillance
- magic AI
- guaranteed revenue
- quote-ready

## Example Prompt For Future Icon Iterations

Use case: `logo-brand`

Asset type: enterprise app icon

Prompt:

```text
Create a minimal modern startup-style app icon for a customer scouting automation agent named Zurich Scout.
The icon should combine a simple lens/radar metaphor with one small gold signal dot, suggesting qualified account discovery.
Use a clean white rounded-square tile, deep Zurich-like blue arcs, dark navy depth, and one restrained gold accent.
No text, no shield, no Zurich logo, no letter Z, no mascot, no complex network, no cyber lock, no gradients beyond subtle depth.
Make it crisp, enterprise credible, calm, and suitable for a Zurich Insurance hackathon prototype.
```
